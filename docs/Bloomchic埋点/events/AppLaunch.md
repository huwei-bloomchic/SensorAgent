# AppLaunch

## 基本信息

- **事件显示名**: APP启动时上报
- **事件英文名**: `AppLaunch`
- **所属模块**: 0.APP特有的埋点
- **应埋点平台**: Android,IOS

## 事件属性

| 属性英文名 | 属性显示名 | 数据类型 | 属性值示例或说明 | 应埋点平台 | 触发时机 | 备注 | 日期 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| `deep_link_value` | APP被链接唤醒的链接 | STRING | App 2.13新增 |  |  |  |  |
| `permission` | 通知权限 | Number | 1 有通知权限<br/>2 没有通知权限<br/>3 没有通知权限，Android 13默认拒绝<br/>4 没有通知权限 不再询问<br/>iOS（1 有通知权限）（2 没有通知权限） |  |  |  |  |
