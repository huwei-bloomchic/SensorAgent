# BigPictureClick

## 基本信息

- **事件显示名**: 点击查看大图
- **事件英文名**: `BigPictureClick`
- **所属模块**: 3.【商详页】的埋点
- **应埋点平台**: Web

## 事件属性

| 属性英文名 | 属性显示名 | 数据类型 | 属性值示例或说明 | 应埋点平台 | 触发时机 | 备注 | 日期 |
