# CartAddClick

## 基本信息

- **事件显示名**: 点击【ADD】
- **事件英文名**: `CartAddClick`
- **所属模块**: 4.【购物车】埋点
- **应埋点平台**: Android,iOS

## 事件属性

| 属性英文名 | 属性显示名 | 数据类型 | 属性值示例或说明 | 应埋点平台 | 触发时机 | 备注 | 日期 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| `source` | 来源 | NUMBER | 0=轮播横幅；1=横幅点开后弹窗；2=优惠券列表； | APP3.6新增 |  | 购物车-券包列表点add 或者轮播消息点add |  |
| `promotion_type` | 促销活动类型 | int | 0=凑运费门槛；1=凑优惠门槛；2=登录； | APP3.6新增 |  |  |  |
| `coupon_code` | 券code | STRING |  | APP3.6新增 |  |  |  |
| `coupon_source` | 优惠券来源 | STRING | 0=平台阶梯券；1=用户券包券； 3=凑运费 | APP3.6新增 |  |  |  |
| `coupon_type` | 优惠券类型 | STRING | 1：百分比优惠，2：金额优惠，3：免运费 4:折扣显示商品个数<br/>和后端返回的int一致 | APP3.6新增 |  |  |  |
| `price` | 门槛金额差额 | NUMBER | 是门槛金额，上报这个值。还需要多少钱达到使用门槛；比如$49.99 | APP3.6新增 |  |  |  |
