# CustomizeCookiesResult

## 基本信息

- **事件显示名**: 隐私协议，用户是否允许数据收集
- **事件英文名**: `CustomizeCookiesResult`
- **所属模块**: 0.全局埋点
- **应埋点平台**: Android,IOS

## 事件属性

| 属性英文名 | 属性显示名 | 数据类型 | 属性值示例或说明 | 应埋点平台 | 触发时机 | 备注 | 日期 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| `page_from` | 页面来源，上一个页面的page_code,首页和设置页 | STRING |  |  |  |  |  |
| `enable_share_data` | 1表示on，0表示off，用户点了ACCEPT ALL也表示on | INT |  |  |  |  |  |
