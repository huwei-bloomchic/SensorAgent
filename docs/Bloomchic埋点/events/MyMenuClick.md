# MyMenuClick

## 基本信息

- **事件显示名**: 个人中心我的菜单点击
- **事件英文名**: `MyMenuClick`
- **所属模块**: 6.【me页】埋点
- **应埋点平台**: Android,IOS

## 事件属性

| 属性英文名 | 属性显示名 | 数据类型 | 属性值示例或说明 | 应埋点平台 | 触发时机 | 备注 | 日期 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| `email` | 登录用户email | STRING | 未登录和登录都要上报，假如登录了，就把登录邮箱属性也上报。 |  |  |  |  |
| `MenuName` | 一级/二级菜单名称 | STRING | CompanyInfo/About us/.../Support Center/Quick Link等所有一二级菜单名。点击一级菜单，就上传一级菜单名，点了二级菜单，就上传二级菜单名。 |  |  |  |  |
