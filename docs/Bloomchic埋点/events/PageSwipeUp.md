# PageSwipeUp

## 基本信息

- **事件显示名**: 上划页面
- **事件英文名**: `PageSwipeUp`
- **所属模块**: 0.全局埋点
- **应埋点平台**: Web

## 事件属性

| 属性英文名 | 属性显示名 | 数据类型 | 属性值示例或说明 | 应埋点平台 | 触发时机 | 备注 | 日期 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| `page_code` | 页面位置 | STRING | 首页、商品列表、商品详情页、结算页、其他 | JavaScript | 上划页面时上报；<br/>单次进入页面中只上报一次 |  |  |
| `page_id` | 页面ID | STRING | 首页/列表ID/商品SPU/结算页 | JavaScript |  |  |  |
