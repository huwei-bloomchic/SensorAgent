# PasswordResetResult

## 基本信息

- **事件显示名**: 密码重置/忘记密码失败（失败才上报）
- **事件英文名**: `PasswordResetResult`
- **所属模块**: 0.全局埋点
- **应埋点平台**: Android,Web,IOS
