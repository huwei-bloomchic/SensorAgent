# SMSsubscribe

## 基本信息

- **事件显示名**: footer点击订阅
- **事件英文名**: `SMSsubscribe`
- **所属模块**: 1.【首页】埋点
- **应埋点平台**: Web

## 事件属性

| 属性英文名 | 属性显示名 | 数据类型 | 属性值示例或说明 | 应埋点平台 | 触发时机 | 备注 | 日期 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| `email_content` | 用户邮箱 | STRING | 当前用户注册得邮箱 |  |  |  |  |
| `sms_content` | 手机号 | STRING | 用户在sms框输入的内容 |  |  |  |  |
| `check_result` | 手机号校验结果 | INT | 1=通过<br/>0=失败 |  |  |  |  |
| `page_code` | 页面位置 | STRING | 首页、商品列表、商品详情页、其他 | JavaScript |  |  |  |
